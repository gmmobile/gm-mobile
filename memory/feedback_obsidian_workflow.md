Sempre que fizer qualquer alteração no projeto GM Móbile, registrar a mudança no vault do Obsidian em `c:\Users\ketll\Desktop\GM-MÓBILE\GM-MÓBILE\`.

**Why:** O usuário quer usar o Obsidian como memória/cérebro do projeto para manter histórico e contexto de tudo que foi feito.

**How to apply:**
- Após cada mudança no site (CSS, HTML, JS), atualizar a nota relevante no vault
- Se for algo novo, criar nota nova e linkar no Bem-vindo.md
- Usar as notas existentes como ponto de partida em cada conversa nova
